=head1 NOMBRE

muttprint - impresión elegante de mensajes de correo con Mutt


=head1 SINOPSIS

muttprint [I<opción>]... [-f I<archivo>]


=head1 DESCRIPCIÓN

Esta página manual documenta brevemente la utilidad B<Muttprint>. Esta 
página manual fue escrita originalmente para la distribución Debian 
GNU/Linux porque el programa original no tenía una página manual, pero
ahora me hice cargo de esta página manual en el formato POD.

B<NOTA:> La ruta F</usr/share/doc/packages/muttprint> podría ser 
F</usr/doc/packages/muttprint> o F</usr/share/doc/muttprint> en tu 
sistema!

B<Muttprint> es una utilidad que da formato a la impresión de Mutt y otros
clientes de correo como XFMail o PINE para parecerse a la impresión de
Netscape Messenger o Kmail. Este puede imprimir un pequeño pingüino en la
primera página y un encabezado en cada página. Además, sólo imprime las
cabeceras más importantes, pero no toda la plétora de ellas.

Por intrucciones de como usar B<Muttprint> consulta 
F</usr/share/doc/packages/muttprint/muttprint-doc-es.html>.

De todos modos, tienes que colocar la siguiente línea en tu F</etc/Muttrc>
o F<~/.muttrc>:

B<set print_command="muttprint">

Si quieres personalizar la configuración de B<Muttprint>, simplemente
copia 
F</usr/share/doc/packages/muttprint/sample-muttprintrc>
a F</etc/Muttprintrc> o F<~/.muttprintrc> con confianza si quieres cambiar 
la configuración de todo el sistema o de un usuario.

B<Muttprint> por defecto usa la configuración del idioma Inglés si la 
variable de entorno I<LANG> no está definida.  Por ejemplo en un entorno 
Español deberías definir: 

B<export LANG=es_ES>

Esto puede ser realizado por ejemplo en tu F<~/.bashrc> local.

Si B<Muttprint> no imprime y muestra el mensaje 
C<lpr: unable to print file: ...>, 
deberías tratar de usar otra impresora diferente a I<lp>, por ejemplo I<lp5> o
I<tp0> (si usas TurboPrint).

B<NOTA PARA LOS USUARIOS DE CUPS:> La configuración B<PRINTER="lp"> no 
funcionará, y tendrás que configurar el nombre de la impresora Cups en su
lugar (por ejemplo, I<LaserJet>), el cual puedes verificar fácilmente
tecleando lpq en la línea de comandos.


=head1 OPCIONES

B<POR FAVOR CONSIDERA:> Estas opciones tienen preferencia sobre las opciones
correspondientes en F<~/.muttprintrc> y F</etc/Muttprintrc>.

Aquí encotrarás la descripción de todas las opciones de línea de comandos (las 
descripciones de las opciones del archivo de configuración está incluído en el
archivo F<sample-muttprintrc> distribuído con B<Muttprint>):

=over 7

=item B<-h, --help>

Muestra en pantalla una lista de todas las opciones posibles.


=item B<-v, --version>

Muestra en pantalla la versión actual de B<Muttprint>.


=item B<-f> I<archivo>B<, --file> I<archivo>

Lee desde <archivo> en lugar de STDIN. Útil para algunos clientes de correo
como I<XFMail>. Si I<file> es I<->, B<Muttprint> lee desde STDIN.


=item B<-p> I<nombreimpresora>B<, --printer> I<nombreimpresora>

Usa otra impresora en lugar de la impresora I<lp> o de la impresora definida
en el archivo de configuración de Muttprint. También puedes 
B<imprimir en un archivo> utilizando F<TO_FILE:/path/to/file> como nombre de
impresora. Si quieres que el resultado vaya a B<STDOUT>, utiliza I<-> como
nombre de impresora.


=item B<-i> I<archivo>B<, --penguin> I<archivo>

Utiliza  F<archivo> como el gráfico a ser impreso en la parte superior derecha
de la hoja. F<archivo> debe contener la ruta completa. Utiliza I<off> como
F<archivo> si no quieres imprimir el gráfico.


=item B<-t> I<número>B<, --speed> I<número>

Intervalo de tiempo en segundos que tu impresora necesita para
imprimir una página. Sólo es necesario si utilizas impresión dúplex. Por
defecto es 30 (segundos)

=item B<-w> I<número>B<, --wait> I<número>

Intervalo de tiempo en segundos que B<Muttprint> debería esperar entre la
impresión de páginas pares e impares para impresión dúplex.

=item B<-F> I<fuente>B<, --font> I<fuente>

Aquí puedes elegir una agradable familia de fuentes para tu impresión. Las
posibles son: I<Latex>, I<Latex-bright>, I<Times>, I<Utopia>, I<Palatino>,
I<Charter> y I<Bookman>. Que todas esas fuentes estén disponibles en tu sistema
depende de tu sistema LaTeX. Por defecto es I<Latex-bright>.


=item B<-H, --headrule> | B<-noH, --noheadrule>

Aquí puedes decidir si se imprimirá o no una línea separadora después de la
línea superior a partir de la página 2.


=item B<-b, --footrule> | B<-nob, --nofootrule>

Aquí puedes decidir si se imprimirá o no una línea separadora sobre el pie de
página.


=item B<-S> I<estilo> | B<--frontstyle> I<estilo>

Elige un estilo para la impresión de las cabeceras del mensaje en la primera
página:
I<plain>, I<border> (por defecto), I<fbox>, I<shadowbox>, I<ovalbox>, I<Ovalbox>,
I<doublebox>, I<grey>, I<greybox>. Consulta el manual para una descripción 
detallada de ésos valores.


=item B<-P> I<formatopapel>B<, --paper> I<formatopapel>

Elige el formato del papel: I<A4> en la mayoría de los países ("DIN A4" en
Alemania) y I<letter> en los U.S.


=item B<-l> I<idioma>B<, --lang> I<idioma>

Tienes que proporcionar la forma corta de tu idioma (actualmente soportados: 
I<de> (Alemán), I<en> (Inglés), I<it> (Italiano) y en particular I<es>
(Español). Por defecto es el idioma definido por la variable de entorno
I<LANG >; si éste no está soportado, por defecto será Inglés.



=item B<-D, --debug> | B<-noD, --nodebug>

Determina si B<Muttprint> escribirá a un archivo log F</tmp/muttprint.log>.
Útil para pedir ayuda a otras personas cuando B<Muttprint> no funciona.


=item B<-d, --duplex> | B<-nod, --noduplex>

Activa o desactiva la impresión dúplex (ambos lados). Si está activada,
B<Muttprint> imprime sólo las páginas impares y espera un tiempo antes de
imprimir las pares.


=item B<-2> | B<-1>

Imprime dos páginas en una carilla como se muestra aquí:

	+----------+----------+
	|          |          |
	|          |          |
	|    1     |     2    |
	|          |          |
	|          |          |
	|          |          |	
	+----------+----------+
	
Esta función es llamada I<modo ahorro de papel> en el manual y en el archivo
de configuración.


=item B<-s, --rem_sig> | B<-nos, --norem_sig>

No imprime la firma (separada del resto del mensaje por C<-- >). Esto es útil
cuando las firmas son muy largas. Pero más y más personas (o clientes de
correo) no separan la firma correctamente, así que esta funcionalidad no
siempre funciona. 


=item B<-q, --rem_quote> | B<-noq, --norem_quote>

No imprime el párrafo citado. B<Muttprint> detecta el texto citado con la
siguiente expresión regular:

	/^([\t]*[|>:}#])+/

Es la misma expresión regular que utiliza B<Mutt>, así que debería funcionar :-)

=back



=head1 ARCHIVOS

=over 7

=item F</etc/Muttprintrc>

Archivo de configuración global del sistema.


=item F<~/.muttprintrc>

Archivo de configuración del usuario.


=item F</tmp/muttprint-XXXXX>

Archivos temporales creados por B<Muttprint>.


=item F</usr/share/doc/packages/muttprint/>

Manuales de B<Muttprint> en diferentes idiomas y formatos.


=item F</usr/share/muttprint/>

Algunas imágenes que pueden utilizarse para ser impresas en la primera página
de tu mensaje de correo.


=back
